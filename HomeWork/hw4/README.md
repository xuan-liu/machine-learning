This assignment is prepared for a Kaggle competition. URL:https://www.kaggle.com/c/uc-irvine-cs-273a-fall-2018

In this compeition, it is intended to predict whether there is rainfall at a location, based on (processed) infrared satellite image information. 

After trying linear classifiers, nearest neighbors, decision trees, neural networks and adjusting their parameters based on a subset of data, I choose decision tree and my leaderboard AUC is 0.72548.
